🆕 Custom
Weighted Reciprocal Rank Fusion (RRF)
search/fusion.py → reciprocal_rank_fusion()
Merges both ranked lists. Docs in BOTH lists get boosted. Semantic weight = 1.0, BM25 weight = 0.4 (semantic dominates).
RRF_score(doc) = Σ weight_i / (60 + rank_i) Semantic: [A(r1), C(r2), F(r3), B(r4)] w=1.0 BM25: [C(r1), A(r2), D(r3), B(r4)] w=0.4 Doc_A: 1.0/(60+1) + 0.4/(60+2) = 0.023 Doc_C: 1.0/(60+2) + 0.4/(60+1) = 0.023 ← both! Doc_D: 0 + 0.4/(60+3) = 0.006 ← BM25 only → top 20 unique docs by RRF score
Weighted RRF k=60
